\# Симулятор жизни



Консольный симулятор экосистемы на Python.

Учебный проект по объектно-ориентированному программированию и работе с Git.



\## Модель



Модель "хищник-жертва" с растениями:



\- \*\*Plant (Растение)\*\* — получает энергию от солнца (+3 в день), размножается при энергии ≥ 15

\- \*\*Herbivore (Травоядное)\*\* — ест растения (+5 энергии), размножается при энергии ≥ 25

\- \*\*Carnivore (Хищник)\*\* — охотится на травоядных (шанс 70%, +8 энергии), размножается при энергии ≥ 35



Каждый день организмы:

1\. Питаются (растения → травоядные → хищники)

2\. Стареют и тратят 1 энергии

3\. Размножаются при достаточной энергии



При энергии ≤ 0 организм умирает и удаляется из экосистемы.
